# Machar-Kotha

Simple Blog Website built with Django.

## Overview

Machar-Kotha is a minimal blog application that demonstrates Django app structure and basic CRUD functionality for posts.

## Features

- Create, read, update, and delete blog posts
- Clean template-based front-end
- Django-powered routing and views

## Setup

1. Create and activate your Python virtual environment.
2. Install dependencies from `requirements.txt` or `django`.
3. Run migrations:
   ```bash
   python manage.py migrate
   ```
4. Start the development server:
   ```bash
   python manage.py runserver
   ```

## Usage

Open `http://127.0.0.1:8000/` in your browser to view the homepage and interact with the blog.

## Notes

This repository is intended as a simple Django blog practice project.
